# T055 — View-layer cleanups: AnyView, GeometryReader, Haptics.prepare()

**Built:** 2026-07-17

## Summary
Three independent, behaviour-preserving cleanups:
1. **AnyView removal**: Made ItemRow generic over its editor type (using QuantityEditor?), eliminating type erasure from the quantity-editor plumbing
2. **Dead GeometryReader removal**: Deleted unused GeometryReader and w/h parameters from Theme.swift's BasketBackground.dots()
3. **Haptics.prepare() deletion**: Removed dead prepare() function (defaulted to deletion per task spec) and updated doc comment

## Changes

### Sources/Views/ItemRow.swift
- Changed `struct ItemRow: View` → `struct ItemRow<Editor: View>: View`
- Changed `var quantityEditor: AnyView?` → `var quantityEditor: Editor?`
- Added convenience initializer extension `ItemRow where Editor == EmptyView` to allow call sites (GotItSection, SnapshotTests) to compile unchanged

### Sources/Views/ToGetSection.swift
- Changed `editor: (GroceryItem) -> AnyView?` → `editor: (GroceryItem) -> QuantityEditor?`

### Sources/Views/ShoppingListView.swift
- Changed `editor(for:)` return type from `AnyView?` → `QuantityEditor?`
- Removed `AnyView()` wrapper; now returns concrete `QuantityEditor` directly

### Sources/Theme/Theme.swift
- Removed `GeometryReader` wrapper from `BasketBackground.body`
- Changed `dots(_ color: Color, _ w: CGFloat, _ h: CGFloat)` → `dots(_ color: Color)`
- Removed unused width/height parameters — Canvas gets its own `size` directly

### Sources/Services/Haptics.swift
- Deleted `prepare()` function (lines 10-15)
- Updated doc comment: `"via prepared generators"` → `"via pre-allocated generators"` (truthful reflection of current implementation)

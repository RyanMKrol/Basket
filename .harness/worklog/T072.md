### Design decisions
- Both add and combined widgets use `widgetURL(URL(string: "basket://add"))` to deep link into the app
- Add widget is small-only since it's primarily a button
- Combined widget is medium-only to fit both list and button; lists fill the space in both sizes on the existing widget
- Reused existing `BasketWidgetProvider` (which reads the shared App Group store) for the combined widget to show live data
- Add widget provider is stateless (returns an empty entry) since it has no dynamic data
- Used `Theme.leaf` for the + button color (green accent) and `Theme.card` for the combined widget's add-button background card

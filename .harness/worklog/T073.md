## Work completed

1. **Created `tools/profile_app.sh`** (executable, `set -euo pipefail`):

   - **CRITICAL attach-reliability guard:** detects empty/Basket-frameless traces and exits NON-ZERO with a message pointing at README. Never masquerades an empty trace as success.

   - Prints human-readable markdown summary to stdout.
   - Echoes exact `xcprof compare` command for A/B testing.
   - Traces/JSON stored under `build/perf-traces/` (covered by existing .gitignore entry on `build/`).
   - `--refresh-baseline` flag copies analyzed JSON to `tools/perf-baseline/baseline.json` (pretty-printed via jq), only succeeding with a non-empty, Basket-attributed trace.

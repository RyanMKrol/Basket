## Overview
Human step: run the app, cycle the three background-blur intensities built in T076, and pick the one that looks best. This is a taste call (which blur reads best against Basket's soft Pastel Dots aesthetic), so it can't be machine-verified — a human chooses, and the follow-up task bakes the choice in.

## Do
Run the app (`./build_run.sh`, or from Xcode) and focus the add bar so the background blur appears. Cycle through the three blur intensities via the temporary switch T076 added. Compare them: the goal is clear foreground/background separation (add bar + suggestions pop, list recedes) WITHOUT the blur feeling heavy or muddy against the dotted background. Pick the winner.

## Done when
- The chosen blur intensity is recorded concretely — WHICH of the three, and its exact value (radius in pt, or which `Material`) — in this task's worklog (or handed directly to whoever builds the follow-up), so the hardcode task can bake in exactly that value with no ambiguity.

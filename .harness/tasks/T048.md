## Overview
Move Basket's pure, dependency-free logic (emoji cascade, suggestions, formatting, measure, seasonality, list logic + the Suggestion value type) into a local SwiftPM package, BasketCore, consumed by the app target. This is step 1 of retiring the hand-rolled native harness: once the pure logic lives in a package, `swift test` can run it natively in a follow-up task. In THIS task the app, all existing test suites, and the (interim) native harness must all stay green.
